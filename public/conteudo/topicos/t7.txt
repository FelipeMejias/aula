7. Aprofundar Loops em Lista

7-A. Separar string
Para separar uma string em uma lista de substrings, usamos o método split().

Podemos manipular o índice de uma lista dentro de um loop.

A forma de acessar o índice diretamente é usando o for ... in (Python) ou for ... of com um contador (JavaScript).
